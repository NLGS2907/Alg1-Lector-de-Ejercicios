8=<

t=**a)** Realizar una función que, dada una lista, devuelva una nueva lista cuyo contenido sea igual a
t=   la original pero invertida.
t=   Así, dada la lista `['Di', 'buen', 'día', 'a', 'papa']`, deberá devolver
tn=   `['papa', 'a', 'día', 'buen', 'Di']`.

t=**b)** Realizar otra función que invierta la lista, pero en lugar de devolver una nueva, modifique
t=   la lista dada para invertirla, **sin** usar listas auxiliares.

9=<
Escribir una función `empaquetar` para una lista, donde empaquetar significa indicar la repetición de valores consecutivos mediante una tupla (valor, cantidad de repeticiones).
Por ejemplo, `empaquetar([1, 1, 1, 3, 5, 1, 1, 3, 3])` debe devolver `[(1, 3), (3, 1), (5, 1), (1, 2), (3, 2)]`.
